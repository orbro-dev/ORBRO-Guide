# 소개

### 여기에서 먼저 확인하세요

* `ORBRO OS 소개`: ORBRO OS가 어떤 서비스인지, 어떤 상황에서 도움이 되는지 안내합니다.
* `주요 기능 한눈에 보기`: 처음 사용할 때 어떤 기능을 먼저 보면 좋은지 빠르게 정리한 페이지입니다.
* `용어 설명`: 문서에서 자주 나오는 용어를 쉽게 풀이한 페이지입니다.

***

### 이렇게 읽으면 좋습니다

1. `ORBRO OS 소개`에서 전체 역할을 먼저 이해합니다.
2. `주요 기능 한눈에 보기`에서 자주 쓰는 기능과 흐름을 확인합니다.
3. 익숙하지 않은 용어가 있으면 `용어 설명`을 참고합니다.
4. 그다음 `시작하기`로 이동해 로그인과 첫 화면을 익힙니다.
