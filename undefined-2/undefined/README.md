# 로그인

ORBRO OS는 다양한 사용자 환경과 보안 요건을 충족하기 위해, 다음의 **세 가지 로그인 방식을 제공합니다.** 사용자는 본인의 등록 정보에 따라 적절한 인증 수단을 선택하여 안전하게 로그인할 수 있습니다.



#### 1. 아이디 기반 로그인

* 사용자가 등록한 고유 사용자 아이디(ID)를 입력하여 로그인합니다.
* 아이디 입력 후 계정이 존재하는 경우에만 비밀번호 입력 단계로 진행됩니다.
* 조직 내부 사용자나 고정 계정 체계를 사용하는 환경에 적합합니다.

#### 2. 이메일 주소 기반 로그인

* 이메일 주소로 로그인하려면, 해당 정보가 사용자 계정에 **사전 등록되어 있어야 합니다.**
* 등록되지 않은 이메일 주소로는 로그인 절차에 접근할 수 없습니다.

#### 3. 휴대폰 번호 기반 로그인

* &#x20;휴대폰 번호를 통해 SMS 인증 방식으로 로그인할 수 있습니다.
* 번호를 입력하면 인증번호가 발송되며, 이를 입력하여 본인 인증을 완료합니다.
* 이 로그인 방식도 **사전에 사용자 정보에 등록된 번호만 사용**할 수 있습니다.
