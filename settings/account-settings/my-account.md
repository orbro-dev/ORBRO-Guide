# 내 계정

### 메뉴 위치

* 좌측 메뉴 > `Launchpad` > `Setting application` > `계정` > `내계정`

### 계정 유형에 따라 수정할 수 있는 항목

#### ORBRO 계정

수정 가능한 항목은 다음과 같습니다.

* 프로필 이미지
* 사용자 이름
* 등록된 이메일 주소
* 등록된 휴대폰 번호

이메일 주소와 휴대폰 번호 항목에는 각각 `변경` 버튼이 표시됩니다.

`변경` 버튼을 클릭한 뒤 새 이메일 주소 또는 새 휴대폰 번호를 입력하면, 입력한 정보로 인증 코드가 전송됩니다. 인증 코드를 입력해 인증을 완료하면 해당 정보를 변경할 수 있습니다.

#### Local 계정

수정 가능한 항목은 다음과 같습니다.

* 프로필 이미지
* 비밀번호

### 사용 방법

1. `내계정` 메뉴를 엽니다.
2. 현재 계정 유형이 ORBRO 계정인지 Local 계정인지 확인합니다.
3. 수정할 항목을 선택합니다.
4. 필요한 정보를 입력하거나 변경합니다.
5. 저장 또는 변경 버튼을 눌러 반영합니다.

### 이메일 주소 또는 휴대폰 번호 변경 방법

1. 변경하려는 항목의 `변경` 버튼을 클릭합니다.
2. 새 이메일 주소 또는 새 휴대폰 번호를 입력합니다.
3. 입력한 정보로 전송된 인증 코드를 확인합니다.
4. 인증 코드를 입력해 본인 인증을 완료합니다.
5. 인증이 완료되면 변경 내용을 저장합니다.

### 변경 전 확인할 내용

* 현재 로그인한 계정이 본인 계정인지 확인합니다.
* 계정 유형에 따라 수정 가능한 항목이 다른지 확인합니다.
* 이메일 주소나 휴대폰 번호를 변경할 때는 인증 코드를 받을 수 있는 상태인지 먼저 확인합니다.

### 참고해 주세요

수정 가능한 항목은 권한과 구축 환경에 따라 다를 수 있습니다.
