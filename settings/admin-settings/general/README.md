# 일반

### 메뉴 위치

* `좌측 메뉴 > Launchpad > Setting application > 관리자 > 일반`

***

### 일반 메뉴에서 주로 하는 일

* 워크스페이스 이름과 로고 관리
* 기본 언어와 시작 위치 설정
* 오브젝트 분류 기준과 타임아웃 시간 관리

***

### 하위 탭 안내

#### 워크스페이스

워크스페이스 이름과 로고처럼 서비스의 대표 정보를 관리하는 탭입니다.

* 로그인 화면에 표시되는 이름과 로고를 관리할 때 사용합니다.
* 조직이나 현장을 구분하는 대표 정보를 정리할 때 확인합니다.

#### 인터페이스

화면 언어와 시작 위치처럼 사용자가 처음 접하는 환경을 설정하는 탭입니다.

* PC 화면의 언어를 바꾸거나 시작 위치를 지정할 때 사용합니다.
* 로그인 후 기본으로 열릴 건물과 공간을 정할 때 확인합니다.

#### 오브젝트

오브젝트 카테고리와 태그 유형별 타임아웃을 관리하는 탭입니다.

* 객체를 어떤 분류 기준으로 볼지 정할 때 사용합니다.
* 신호가 끊긴 객체를 언제 오프라인으로 처리할지 조정할 때 확인합니다.

***

### 이렇게 보면 이해하기 쉽습니다

1. 서비스 이름과 로고를 먼저 맞추려면 `워크스페이스`를 확인합니다.
2. 언어와 시작 화면을 정하려면 `인터페이스`를 확인합니다.
3. 객체 분류와 오프라인 처리 기준을 정하려면 `오브젝트`를 확인합니다.

***

### 참고해 주세요

* 일반 메뉴의 설정은 여러 사용자 화면에 공통으로 영향을 줄 수 있습니다.
* 변경 후에는 로그인 화면, 시작 위치, 오브젝트 표시 방식이 의도한 대로 반영되는지 함께 확인하는 것이 좋습니다.
