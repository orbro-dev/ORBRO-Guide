# ORBRO 계정 사용자 등록

### 메뉴 위치

* `좌측 메뉴 > Launchpad > Setting application > 관리자 > 사용자 > 사용자 추가`

***

### ORBRO 계정 사용자 등록이란?

* ORBRO 계정은 `이메일 주소` 또는 `휴대폰 번호` 기반으로 사용하는 계정입니다.
* 사용자는 ORBRO 계정 정보로 로그인합니다.
* Local 계정과 달리 관리자가 별도의 `아이디 / 비밀번호`를 직접 만드는 방식이 아닙니다.

***

### 등록할 때 함께 확인하는 항목

* 사용자 이름
* 연락처
* 그룹(선택)

#### 연락처

프로젝트 운영 방식에 따라 `이메일 주소` 또는 `휴대폰 번호` 중 하나를 기준으로 사용자를 등록합니다.

* 이메일 주소 기반 사용자인지
* 휴대폰 번호 기반 사용자인지
* 현재 운영 중인 ORBRO 계정 체계와 맞는지

를 먼저 확인하는 것이 좋습니다.

#### 권한

* ORBRO 계정 사용자는 등록할 때 권한을 설정하지 않습니다.
* 권한은 사용자 등록 후 필요에 따라 수정할 수 있습니다.

***

### Local 계정 등록과 다른 점

* `아이디`를 직접 만들지 않습니다.
* `비밀번호`를 직접 입력해 생성하지 않습니다.
* ORBRO 계정 정보 기준으로 사용자를 등록하고, 이후 해당 계정 방식으로 로그인합니다.

***

### 등록 순서

1. `사용자` 메뉴를 엽니다.
2. `사용자 추가`를 클릭합니다.
3. 계정 유형을 확인하고 ORBRO 계정 기준으로 등록을 진행합니다.
4. 사용자 이름을 입력하고, 연락처로 사용할 `이메일 주소` 또는 `휴대폰 번호`를 입력합니다.
5. 필요하면 `그룹(선택)`을 지정합니다.
6. 저장 후 사용자 목록에서 계정 타입과 그룹이 올바르게 표시되는지 확인합니다.
7. 권한이 필요하면 등록 후 사용자 정보에서 수정합니다.

***

### 등록 후 확인해 주세요

* 사용자 목록의 `계정 타입`이 올바른지 확인합니다.
* 그룹이 의도한 운영 범위와 맞는지 확인합니다.
* 사용자가 이메일 주소 또는 휴대폰 번호 기반 로그인 흐름으로 접속 가능한지 확인합니다.
* 권한이 필요하면 등록 후 별도로 수정합니다.

***

### 참고해 주세요

* ORBRO 계정 사용자는 ORBRO 계정 체계를 사용하는 사용자에게 적합합니다.
* 운영 환경에 따라 이메일 주소 기반 또는 휴대폰 번호 기반 사용 비율이 다를 수 있으므로, 등록 전에 내부 운영 기준을 먼저 확인하는 것이 좋습니다.
