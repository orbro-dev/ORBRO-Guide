# 사용자

### 메뉴 위치

* `좌측 메뉴 > Launchpad > Setting application > 관리자 > 사용자`

***

### 화면 구성

#### 상단 탭

* `사용자`: 사용자 계정을 조회하고 등록하는 탭입니다.
* `그룹`: 사용자 그룹을 조회하고 관리하는 탭입니다.

#### 사용자 탭 목록

사용자 목록에서는 다음 항목을 중심으로 확인합니다.

* `사용자 이름`
* `권한`
* `계정 타입`
* `그룹`

목록 오른쪽의 더보기 메뉴를 통해 개별 사용자 관리 작업을 이어갈 수 있습니다.

#### 상단 기능

* `검색`: 사용자 이름 등으로 목록을 빠르게 찾을 때 사용합니다.
* `엑셀 다운로드`: 현재 사용자 목록을 파일로 내려받아 정리하거나 공유할 때 사용합니다.
* `사용자 추가`: 신규 사용자를 등록할 때 사용합니다.

***

### 사용자 메뉴에서 할 수 있는 일

* 사용자 목록 조회
* 사용자 검색
* 사용자 목록 엑셀 다운로드
* 사용자 추가
* 사용자 정보 수정
* 권한 확인
* 그룹 연결 상태 확인

***

### 계정 유형별 사용자 등록 방식

사용자 등록 방식은 `ORBRO 계정`과 `Local 계정`에 따라 다릅니다.

#### ORBRO 계정 사용자

* 이메일 주소 또는 휴대폰 번호 기반 계정 체계를 사용하는 사용자입니다.
* Local 계정처럼 아이디와 비밀번호를 직접 만드는 방식이 아닙니다.
* 자세한 등록 방법은 아래 페이지를 참고해 주세요.

#### Local 계정 사용자

* 아이디와 비밀번호를 직접 생성하는 사용자입니다.
* 관리자가 사용자 이름, 아이디, 권한, 그룹, 비밀번호를 입력해 계정을 등록합니다.
* 자세한 등록 방법은 아래 페이지를 참고해 주세요.

### 이렇게 사용합니다

1. `사용자` 탭에서 현재 등록된 사용자를 확인합니다.
2. 필요한 사용자를 검색해 현재 권한과 계정 타입을 확인합니다.
3. 목록 정리나 공유가 필요하면 `엑셀 다운로드`로 사용자 목록을 내려받습니다.
4. 신규 사용자를 등록할 때는 먼저 `ORBRO 계정`인지 `Local 계정`인지 구분합니다.
5. 계정 유형에 맞는 방식으로 사용자를 등록합니다.
6. 등록 후 목록에서 이름, 권한, 계정 타입, 그룹이 올바르게 표시되는지 확인합니다.

***

### 운영 팁

* 신규 사용자는 먼저 필요한 권한 범위를 정한 뒤 등록하는 것이 좋습니다.
* ORBRO 계정과 Local 계정을 혼동하면 로그인 안내가 달라질 수 있으므로 등록 전에 계정 유형을 먼저 확인해 주세요.
* 퇴사자나 미사용 계정은 즉시 비활성화해 계정 정리를 유지하는 것이 좋습니다.
