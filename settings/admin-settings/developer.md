# 개발자

### 메뉴 위치

* `좌측 메뉴 > Launchpad > Setting application > 관리자 > 개발자`

***

### 이 메뉴에서 확인할 수 있는 항목

#### REST API

* 토큰 발급
* 토큰 이름
* 만료 기간 설정

#### MQTT

* 계정 생성
* 아이디
* 비밀번호

***

### 이 메뉴에서 할 수 있는 일

* 외부 연동을 위한 REST API 토큰 관리
* MQTT 계정 생성 및 관리
* 시스템 연동을 위한 개발자용 설정 확인

***

### 참고해 주세요

개발자 메뉴는 일반 사용자용 메뉴가 아니며, 관리자 또는 연동 담당자만 접근하는 것을 권장합니다.
