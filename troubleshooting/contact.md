# 문의 방법

### 전달하면 좋은 정보

* 사용 중인 워크스페이스 이름
* 로그인한 계정 정보
* 문제가 발생한 메뉴 또는 Application 이름
* 발생 시각
* 어떤 작업을 하다가 문제가 생겼는지
* 화면 캡처 또는 오류 메시지
* 관련 장비 이름 또는 ID(장비 문제인 경우)

### 문의 전 먼저 확인할 문서

* `로그인 문제`
* `권한 문제`
* `장비 연결 문제`
* `알림/조건 설정 문제`

### 운영 팁

문제 상황을 재현할 수 있는 순서를 함께 전달하면 해결 속도가 빨라집니다.

예를 들어 `로그인 > Dashboard 실행 > 특정 보드 선택 > 저장 시 오류 발생`처럼 정리해 주세요.
