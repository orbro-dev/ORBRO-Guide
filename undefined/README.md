# 시작하기

### 여기에서 할 수 있는 일

* 본인 계정에 맞는 로그인 방법을 확인합니다.
* 로그인 후 처음 보게 되는 화면 구성을 이해합니다.
* 상단 바를 기준으로 앱을 실행하고 이동하는 방법을 익힙니다.

***

### 이렇게 읽으면 좋습니다

1. `로그인 개요`를 읽고 본인에게 맞는 로그인 방식을 확인합니다.
2. 해당 로그인 방식을 자세히 안내한 문서를 확인합니다.
3. `첫 화면 이해하기`에서 상단 바와 기본 화면 구성을 익힙니다.
4. `기본 화면 이동 방법`에서 자주 쓰는 이동 흐름을 확인합니다.
