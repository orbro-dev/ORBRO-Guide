# 네비게이션 로고 설정

### 사용 방법

1. 네비게이션 우측 상단에 있는 프로필 아이콘을 클릭합니다.
2. 메뉴에서 \[설정] 항목을 선택하여 설정 페이지로 이동합니다.
3. 설정 페이지 내 \[일반] > \[워크스페이스] 탭을 클릭합니다.
4. ‘네비게이션 로고’ 항목 오른쪽에 있는 \[변경] 버튼을 클릭합니다.
5. 변경하고자 하는 로고 이미지를 업로드합니다.
