# 건물 추가

### 사용 방법

1. 화면 좌측 상단의 \[건물 선택] 버튼을 클릭합니다. (최초 추가시 ORBRO Map 항목으로 설정 되어있습니다.)&#x20;
2. 표시되는 컨텍스트 메뉴 하단의 \[건물 추가] 항목을 클릭합니다.
3. 팝업창이 열리면 건물 이름을 입력하고 \[추가] 버튼을 클릭합니다.
   * 예: U-Space 1B 빌딩
4. 입력이 완료되면 새로운 건물이 목록에 추가됩니다.
