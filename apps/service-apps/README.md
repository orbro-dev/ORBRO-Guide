# 서비스 애플리케이션

### 이런 때 먼저 봅니다

* 비상 상황이나 예외 상황에 대한 대응 절차를 확인해야 할 때
* 담당자 간 동일한 운영 절차를 확인해야 할 때
* 조건 발생 후 어떤 SOP가 생성되었는지 점검할 때

***

### 포함된 앱

* `SOP`: 조건과 연결된 표준 대응 절차를 확인하는 앱
