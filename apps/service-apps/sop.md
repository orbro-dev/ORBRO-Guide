# SOP

### 메뉴 위치

* `애플리케이션` > `SOP`

***

### 사용 전에 확인해 주세요

* SOP는 `조건` 메뉴에서 만든 조건과 연결해 사용합니다.
* AI Event나 다른 애플리케이션에서 직접 SOP를 실행하는 방식은 아닙니다.
* SOP를 새로 만들려면 먼저 기준이 될 조건과 담당 그룹이 준비되어 있어야 합니다.

***

### 이런 때 사용합니다

* 조건 발생 후 어떤 SOP가 생성되었는지 확인하고 싶을 때
* 진행 중인 SOP와 종료된 SOP를 구분해 관리하고 싶을 때
* 현장 대응 절차를 단계별 시나리오로 구성하고 싶을 때
* 위치, 담당자, 발생 시간을 기준으로 SOP 이력을 점검하고 싶을 때

***

### 주요 메뉴와 화면

* `기록`: 생성된 SOP 이력을 조회합니다.
* `설정`: SOP 운영에 필요한 기준과 시나리오를 관리합니다.
* `상단 필터`: `조회 기간`, `상태`, `SOP` 기준으로 결과를 좁혀 봅니다.
* `기록 목록`: `발생 시간`, `SOP 이름`, `상태`, `종료 시간`, `위치`, `담당자`를 확인합니다.
* `설정 목록`: `SOP 이름`, `기준 조건`, `담당자 그룹`, `편집`, `활성화` 상태를 확인합니다.

***

### SOP 추가 시나리오

#### 1. 설정 화면 열기

1. `SOP` 앱을 실행합니다.
2. 좌측 메뉴에서 `설정`을 선택합니다.
3. 우측 상단의 `추가` 버튼을 클릭합니다.
4. `SOP 추가` 팝업이 열리면 왼쪽은 `시나리오`, 오른쪽은 `기본 정보`를 입력하는 구조로 진행됩니다.

#### 2. 기본 정보 입력

기본 정보 영역에서는 아래 항목을 먼저 설정합니다.

* `SOP 이름`: SOP를 구분하는 이름입니다.
* `기준 조건`: SOP를 발생시키는 조건입니다.
* `담당 그룹`: 알림과 대응을 맡을 그룹입니다.
* `SMS 메시지 전송`: 필요 시 SMS 발송 기능을 켤 수 있습니다.

기본 정보가 준비되면 왼쪽 시나리오 영역에서 세부 단계들을 추가합니다.

#### 3. 시나리오 추가

시나리오 영역 상단의 `+ 추가`를 누르면 추가할 수 있는 시나리오 유형이 표시됩니다.

* `객관식`
* `주관식`
* `날짜 및 시간 입력`
* `SMS 발송`
* `안내 이미지`
* `카메라 확인`

추가한 시나리오는 목록으로 쌓이며, 각 항목은 편집하거나 삭제할 수 있습니다.

#### 4. 시나리오 유형별 입력 예시

**객관식**

현장에서 선택형 응답을 받아야 할 때 사용하는 단계입니다.

* 시나리오 이름 입력
* 질문 문구 입력
* 응답 항목 추가
* 필요 시 `복수 선택` 설정

예: `상황 확인` 단계에서 `예 / 아니오` 중 하나를 선택하도록 구성

**주관식**

상황 설명이나 원인 입력이 필요한 단계입니다.

* 시나리오 이름 입력
* 안내 문구 또는 질문 입력
* 사용자가 직접 텍스트를 입력하도록 구성

예: `발생 사유 확인` 단계에서 상황 발생 원인을 직접 작성하도록 구성

**날짜 및 시간 입력**

발생 시각이나 확인 시각을 기록할 때 사용하는 단계입니다.

* 시나리오 이름 입력
* 안내 문구 입력
* 날짜 선택
* 시간 선택

예: `상황 발생 시간 입력` 단계에서 실제 발생 시점을 기록하도록 구성

**SMS 발송**

지정한 대상에게 문자 전송을 수행하는 단계입니다.

* 시나리오 이름 입력
* 안내 문구 또는 메시지 설명 입력
* 전송할 메시지 내용 입력
* `받을 대상` 그룹 선택
* `SMS 전송` 버튼으로 발송 동작 확인

예: `화재 담당자에게 문자 발송` 단계에서 담당 그룹에 즉시 안내

**안내 이미지**

대피도나 안내 이미지를 보여 주는 단계입니다.

* 시나리오 이름 입력
* 설명 문구 입력
* `jpg`, `png` 이미지 업로드
* 미리보기에서 이미지 확인

예: `비상 대피안내` 단계에서 층별 대피 안내 이미지를 보여 주도록 구성

**카메라 확인**

현장 영상을 확인해야 할 때 사용하는 단계입니다.

* 시나리오 이름 입력
* 설명 문구 입력
* 확인할 카메라 선택
* 필요 시 여러 카메라를 추가해 구성

예: `CCTV 영상 확인` 단계에서 특정 구역 카메라를 연결

#### 5. 저장과 운영

1. 필요한 시나리오를 모두 추가합니다.
2. 시나리오 순서와 내용을 다시 확인합니다.
3. `추가` 또는 `저장`으로 SOP를 등록합니다.
4. 설정 목록에서 새 SOP가 정상적으로 보이는지 확인합니다.
5. 필요하면 활성화 스위치로 사용 여부를 조정합니다.

***

### 기본 사용 흐름

1. `SOP` 앱을 실행합니다.
2. `설정`에서 SOP를 추가하거나 기존 SOP를 편집합니다.
3. `기록`으로 이동해 발생 이력을 조회합니다.
4. `조회 기간`, `상태`, `SOP` 필터를 설정합니다.
5. 목록에서 원하는 기록의 발생 시간, 위치, 담당자, 상태를 확인합니다.
6. 필요하면 기록을 정리하거나 삭제합니다.

***

### 함께 보면 좋은 앱

* `조건`

### 참고해 주세요

* 조회 조건에 맞는 기록이 없으면 `기록 없음` 안내가 표시됩니다.
* 진행 중인 SOP가 오래 남아 있거나 담당자가 비어 있으면 운영 체계를 함께 점검해 주세요.
* 시나리오 이름은 현장 담당자가 바로 이해할 수 있는 표현으로 짧고 명확하게 작성하는 것이 좋습니다.
