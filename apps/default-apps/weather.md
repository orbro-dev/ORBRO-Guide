# Weather

### 메뉴 위치

* `애플리케이션` > `Weather`

***

### 사용 전에 확인해 주세요

* 공공데이터 포털(`data.go.kr`)에 가입해야 합니다.
* 사용할 날씨 API를 신청하고 API 키를 발급받아야 합니다.
* 발급받은 API 키를 Weather 앱의 설정 화면에 등록해야 합니다.

***

### 이런 때 사용합니다

* 현재 날씨와 온도를 운영 참고 정보로 확인하고 싶을 때
* 시간대별 날씨 변화를 미리 보고 싶을 때
* 특정 지역 기준 날씨 정보를 조회하고 싶을 때

***

### 주요 메뉴와 화면

* `날씨 현황` 탭: 현재 날씨, 온도, 최고/최저 기온, 시간대별 변화를 확인합니다.
* `설정` 탭: 조회할 지역과 API 키를 등록합니다.
* `API 키 관리`: 등록된 API 키를 저장하거나 삭제합니다.

***

### 기본 사용 흐름

1. 공공데이터 포털에서 날씨 API 키를 발급받습니다.
2. `Weather` 앱을 실행합니다.
3. `설정` 탭에서 조회할 지역을 선택합니다.
4. 발급받은 API 키를 입력하고 저장합니다.
5. `날씨 현황` 탭에서 현재 날씨와 시간대별 변화를 확인합니다.
6. 필요하면 설정을 다시 조정합니다.

***

### 함께 보면 좋은 앱

* `Dashboard`
* `위젯`

***

### 참고해 주세요

* API 키가 없거나 잘못 입력되면 날씨 정보가 표시되지 않을 수 있습니다.
* Weather 정보는 운영 판단을 돕는 참고 정보로 활용하는 것이 좋습니다.
