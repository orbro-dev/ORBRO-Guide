# 기본 애플리케이션

### 이런 때 먼저 봅니다

* 처음 ORBRO OS에 접속해 전체 현황을 확인할 때
* 장비 상태를 점검하거나 등록된 장비를 관리할 때
* 계정이나 관리자 설정을 변경할 때
* 공간 구조를 2D보다 직관적으로 확인하고 싶을 때
* 운영에 참고할 외부 정보가 필요할 때

***

### 포함된 앱

* `Dashboard`: 운영 현황을 한눈에 확인하는 시작 화면
* `Device Manager`: ORBRO OS에 연결된 장비를 유형별로 관리하는 앱
* `Setting`: 계정과 관리자 설정을 관리하는 앱
* `3D View`: 공간 구조를 입체적으로 확인하는 앱
* `Weather`: 운영에 참고할 수 있는 날씨 정보를 확인하는 앱

***

### 먼저 읽기 좋은 순서

1. Dashboard
2. Device Manager
3. Setting
4. 3D View
5. Weather
