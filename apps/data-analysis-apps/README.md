# 데이터 분석 애플리케이션

### 이런 때 먼저 봅니다

* 감지 데이터 수집 상태를 분석하고 싶을 때
* 날짜별 변화나 시간대별 패턴을 비교하고 싶을 때
* 누적 데이터를 바탕으로 운영 품질을 점검하고 싶을 때

***

### 포함된 앱

* `Scan Insight`: 날짜별 스캔 수신 현황과 시간대별 추이를 분석하는 앱
* `Nearby Search`: 등록된 Tag의 감지 상태를 날짜별·시간대별로 분석하는 앱
