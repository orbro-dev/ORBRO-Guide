# 관리자 설정

### 관리자 설정에서 주로 하는 일

* 워크스페이스 운영 기준 확인
* 사용자와 그룹 관리
* 알림 설정
* 조건 생성과 관리
* 개발 연동 정보 확인

***

### 하위 메뉴

* `일반`
* `사용자`
* `알림`
* `조건`
* `개발자`

***

### 사용 순서

1. 관리자 권한으로 로그인합니다.
2. 필요한 설정 메뉴를 선택합니다.
3. 현재 설정값을 확인합니다.
4. 필요한 항목을 수정한 뒤 저장합니다.
5. 변경 사항이 실제 운영 화면에 미치는 영향을 함께 확인합니다.

***

### 참고해 주세요

관리자 설정은 여러 사용자와 운영 화면에 영향을 줄 수 있으므로, 변경 전에 적용 범위를 먼저 확인하는 것이 좋습니다.
