# Local 계정 사용자 등록

### 메뉴 위치

* `좌측 메뉴 > Launchpad > Setting application > 관리자 > 사용자 > 사용자 추가`

***

### Local 계정 사용자 등록이란?

* Local 계정은 `아이디`와 `비밀번호`를 사용하는 계정입니다.
* 관리자가 사용자 정보를 직접 입력해 계정을 생성합니다.
* 로그인 시 사용자는 등록된 아이디와 비밀번호를 사용합니다.

***

### 입력 항목

Local 계정 추가 화면에서는 아래 항목을 입력합니다.

* 프로필 이미지
* `사용자 이름`
* `아이디(영문필수 숫자 조합)`
* `권한`
* `그룹(선택)`
* `비밀번호`
* `비밀번호 확인`

***

### 항목별 설명

#### 프로필 이미지

* 상단의 이미지 영역에서 사용자 프로필 이미지를 등록할 수 있습니다.
* 이미지를 등록하지 않으면 기본 아이콘으로 표시될 수 있습니다.

#### 사용자 이름

* 사용자 목록에서 표시되는 이름입니다.
* 운영자가 해당 사용자를 구분하기 쉬운 이름으로 입력하는 것이 좋습니다.

#### 아이디

* 로그인에 사용하는 Local 계정 아이디입니다.
* 화면 안내 기준으로 `영문 필수`이며 숫자 조합 형태로 입력합니다.

#### 권한

* 사용자에게 부여할 권한 수준을 선택합니다.
* 어떤 메뉴까지 접근할 수 있는지에 영향을 줄 수 있습니다.

#### 그룹(선택)

* 필요한 경우 사용자를 특정 그룹에 연결합니다.
* 그룹을 지정하지 않으면 기본 상태로 운영할 수 있습니다.

#### 비밀번호 / 비밀번호 확인

* 사용자가 로그인할 때 사용할 비밀번호를 입력합니다.
* 두 입력값이 일치해야 등록을 진행할 수 있습니다.

***

### 등록 순서

1. `사용자` 메뉴를 엽니다.
2. `사용자 추가`를 클릭합니다.
3. `Local 계정 추가` 팝업에서 프로필 이미지와 기본 정보를 입력합니다.
4. `권한`을 선택합니다.
5. 필요하면 `그룹(선택)`에서 그룹을 지정합니다.
6. 비밀번호와 비밀번호 확인을 입력합니다.
7. `추가` 버튼을 눌러 계정을 등록합니다.

***

### 등록 후 확인해 주세요

* 사용자 목록에 이름, 권한, 계정 타입, 그룹이 올바르게 표시되는지 확인합니다.
* 사용자가 등록한 아이디와 비밀번호로 실제 로그인 가능한지 확인합니다.
* 그룹과 권한이 운영 목적에 맞게 적용되었는지 점검합니다.

***

### 참고해 주세요

* Local 계정은 내부 운영용이나 별도 계정 체계가 필요한 환경에서 주로 사용합니다.
* 비밀번호는 운영 정책에 맞게 관리하고, 불필요한 공유는 피하는 것이 좋습니다.
