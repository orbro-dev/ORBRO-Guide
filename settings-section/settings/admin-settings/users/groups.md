# 그룹 관리

### 메뉴 위치

* `좌측 메뉴 > Launchpad > Setting application > 관리자 > 사용자 > 그룹 관리`

***

### 그룹 관리에서 할 수 있는 일

* 그룹 생성
* 그룹 이름 및 설명 수정
* 그룹에 사용자 추가 또는 제외
* 그룹별 권한 범위 정리

***

### 사용 방법

1. 좌측 메뉴에서 `Launchpad`를 클릭합니다.
2. `Setting application`을 실행합니다.
3. `관리자 > 사용자 > 그룹 관리`를 선택합니다.
4. 그룹 목록에서 기존 그룹을 확인합니다.
5. 새 그룹이 필요하면 추가합니다.
6. 그룹 이름과 목적을 입력합니다.
7. 해당 그룹에 포함할 사용자를 지정합니다.
8. 권한 범위와 메뉴 노출 정책을 확인합니다.

***

### 그룹을 나누는 기준 예시

* 관리자 / 일반 사용자
* 현장 운영 / 관제 / 보안 담당
* 건물 또는 사이트별 운영 조직

***

### 참고해 주세요

그룹 설정은 사용자 권한 문제를 해결할 때 함께 확인해야 하는 핵심 항목입니다.
