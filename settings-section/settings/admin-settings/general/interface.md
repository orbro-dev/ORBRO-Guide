# 인터페이스

### 메뉴 위치

* `좌측 메뉴 > Launchpad > Setting application > 관리자 > 일반 > 인터페이스`

***

### 이 탭에서 할 수 있는 일

* 언어 설정
* 시작 위치 설정

***

### 언어 설정

언어 설정에서는 ORBRO OS 화면에 표시할 기본 언어를 선택합니다.

* 이 설정은 `PC 환경`에만 적용됩니다.
* 모바일 앱은 기기의 언어 설정을 따릅니다.
* 언어를 바꿔도 즉시 반영되지 않을 수 있으며, `새로고침 후 적용`될 수 있습니다.
* 선택한 언어가 지원되지 않는 일부 서비스는 기본 언어인 `영어`로 표시될 수 있습니다.

#### 확인 방법

1. `언어 설정` 항목을 확인합니다.
2. 오른쪽 드롭다운에서 사용할 언어를 선택합니다.
3. 변경 후 필요한 경우 화면을 새로고침합니다.
4. 주요 화면이 원하는 언어로 표시되는지 확인합니다.

***

### 시작 위치 설정

시작 위치 설정에서는 로그인 후 기본으로 열릴 건물과 공간을 지정합니다.

* 설정한 건물과 공간은 ORBRO OS 접속 시 자동으로 시작 위치로 로딩됩니다.
* 자주 사용하는 현장이나 층을 시작 위치로 지정하면 초기 진입 시간이 줄어듭니다.
* 설정된 공간이 삭제되면 기본 `ORBRO Map`이 시작 위치로 지정됩니다.

#### 설정 방법

1. `시작 위치 설정` 항목을 확인합니다.
2. 건물 드롭다운에서 시작 건물을 선택합니다.
3. 공간 드롭다운에서 시작 공간 또는 층을 선택합니다.
4. 저장 후 다시 로그인하거나 화면을 새로 열었을 때 선택한 위치로 시작하는지 확인합니다.

***

### 변경 전에 확인해 주세요

* 언어 설정은 공통 사용 환경에 영향을 줄 수 있습니다.
* 시작 위치는 사용자가 가장 먼저 보게 되는 화면에 영향을 줍니다.
* 실제로 자주 사용하는 공간을 기준으로 정하면 운영 효율이 높아집니다.

***

### 참고해 주세요

* 언어 변경 후 일부 문구는 새로고침 뒤 반영될 수 있습니다.
* 시작 위치로 지정한 공간이 유지되고 있는지 주기적으로 확인하면 혼선을 줄일 수 있습니다.
