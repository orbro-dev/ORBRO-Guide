# Tag 관리

### 메뉴 위치

* `애플리케이션` > `Device Manager` > `Tag`

***

### Tag 관리에서 할 수 있는 일

* 등록된 Tag 목록 확인
* Tag 상태와 배터리 잔량 확인
* Tag 유형과 MAC Address 확인
* 설치 또는 사용 위치 확인
* 자동 검색된 Tag 선택 후 등록
* 수동으로 Tag 정보 입력 후 등록
* CSV 파일로 여러 Tag 일괄 등록

***

### Tag 목록에서 확인하는 항목

* `Device Name`: 운영 화면에서 구분할 Tag 이름입니다.
* `Status`: `Online`, `Offline`, `No Signal` 같은 상태를 확인할 수 있습니다.
* `타임아웃`: Tag 신호 유효 시간 기준을 확인할 수 있습니다.
* `Device Type`: BLE Tag, UWB Tag, ORBRO AoA Tag 등 Tag 유형을 확인합니다.
* `MAC Address`: 장비 식별에 사용하는 주소입니다.
* `배터리 잔량`: Tag 배터리 상태를 확인합니다.
* `Location`: 현재 장비가 연결되거나 운영 중인 위치를 확인합니다.
* `Last Seen`: 최근 감지된 시점을 확인합니다.

***

### Tag 추가 방법

Tag 추가는 `자동` 방식과 `수동` 방식으로 나뉩니다.

#### 자동 추가

자동 추가는 주변에서 검색되거나 시스템에서 인식한 Tag 장비를 선택해 등록하는 방식입니다.

**자동 추가에서 할 수 있는 일**

* 검색된 Tag 목록 확인
* 여러 Tag를 한 번에 선택해 등록
* 검색 기능으로 원하는 Tag 찾기
* 필터로 특정 제조사 또는 장비 조건만 보기

**자동 추가 방법**

1. `기기 추가`를 클릭합니다.
2. `자동` 탭에서 등록할 Tag를 선택합니다.
3. 필요한 경우 검색이나 필터를 사용해 대상 장비를 좁혀 봅니다.
4. 선택을 마친 뒤 `추가`를 클릭합니다.

#### 수동 추가

수동 추가는 Tag 정보를 직접 입력해 등록하는 방식입니다.

**입력 항목**

* `기기 이름`
* `Device Type`
* `MAC Address`

**선택 가능한 Tag 유형**

* `BLE Tag`
* `UWB Tag`
* `ORBRO AoA Tag`
* `ORBRO Tag`
* `Coin Tag`
* `Slim Tag`
* `Sticker Tag`

**수동 추가 방법**

1. `기기 추가`를 클릭합니다.
2. `수동` 탭으로 이동합니다.
3. `기기 이름`, `Device Type`, `MAC Address`를 입력합니다.
4. 여러 개를 한 번에 등록해야 하면 행을 추가해 계속 입력합니다.
5. 입력을 마친 뒤 `추가`를 클릭합니다.

***

### CSV 업로드

여러 Tag를 한 번에 등록해야 할 때는 CSV 업로드를 사용할 수 있습니다.

#### CSV 업로드 방법

1. 수동 등록 화면에서 CSV 업로드 영역을 확인합니다.
2. 제공되는 양식을 내려받아 필요한 내용을 입력합니다.
3. 작성한 CSV 파일을 업로드합니다.
4. 업로드 내용을 확인한 뒤 등록을 진행합니다.

#### 알아두면 좋은 점

* 한 번에 업로드할 수 있는 Tag 수에는 제한이 있을 수 있습니다.
* 업로드 가능한 파일 형식은 `CSV`입니다.
* 양식 구조를 바꾸지 않고 내용만 편집해 사용하는 것이 좋습니다.

***

### 사용 방법

1. `Tag` 페이지로 이동합니다.
2. 목록에서 `Status`, `Device Type`, `배터리 잔량`, `Last Seen`을 먼저 확인합니다.
3. 신규 Tag가 필요하면 `기기 추가`를 클릭합니다.
4. 자동 검색 장비를 쓸지, 직접 입력할지 선택합니다.
5. 등록 후 목록에 장비가 표시되는지와 상태가 정상인지 다시 확인합니다.

### 참고해 주세요

* Tag 이름이나 유형 정보가 실제 운영 기준과 다르면 위치 기반 화면에서 대상을 구분하기 어려울 수 있습니다.
* `Offline`이나 `No Signal` 상태가 계속 보이면 배터리 상태, 설치 위치, 수신 장비 상태를 함께 확인해 주세요.
* Tag 정보가 정확하지 않으면 위치추적 결과나 검색 결과가 달라질 수 있으므로 등록 후 반드시 목록에서 다시 확인해 주세요.
