# 장비 관리

### 이 챕터에서 확인할 내용

* Device Manager의 기본 사용법
* 기기 상세 화면과 설정 화면을 읽는 방법
* 장비 유형별 목록과 상태 확인 방법
* 등록, 수정, 점검 시 확인해야 할 항목

***

### 권장 읽기 순서

1. `Device Manager 소개`
2. `기기 상세 화면 이해하기`
3. 현재 사용 중인 장비 유형 페이지
4. 현장에 실제 설치된 장비의 세부 설정
