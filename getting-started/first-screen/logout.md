# 로그아웃

### 메뉴 위치

* 상단 바 > `프로필` > `로그아웃`

***

### 이런 때 사용합니다

* 현재 작업을 마치고 ORBRO OS 사용을 종료할 때
* 공용 장비에서 다른 사용자가 이어서 로그인해야 할 때
* 현재 계정으로 열린 세션을 안전하게 종료하고 싶을 때

***

### 주요 메뉴와 화면

#### 프로필 메뉴

* 상단 바에서 프로필 아이콘을 누르면 프로필 메뉴가 열립니다.
* 프로필 메뉴 상단에는 현재 로그인한 사용자의 프로필 이미지, 사용자 이름, 권한 정보가 표시됩니다.

#### 로그아웃 버튼

* 프로필 메뉴 아래에서 `로그아웃` 버튼을 확인할 수 있습니다.
* 로그아웃 버튼을 누르면 현재 로그인 상태를 종료할 수 있습니다.

***

### 기본 사용 흐름

1. 상단 바에서 `프로필` 아이콘을 누릅니다.
2. 프로필 메뉴에서 현재 로그인한 사용자 정보를 확인합니다.
3. 아래의 `로그아웃` 버튼을 누릅니다.
4. 현재 계정 사용을 종료합니다.

***

### 참고해 주세요

* 공용 PC나 공동 운영 환경에서는 작업이 끝난 뒤 로그아웃하는 것이 좋습니다.
* 로그아웃 전에 필요한 설정이나 작업 내용을 먼저 확인하고 마무리하는 것이 좋습니다.
